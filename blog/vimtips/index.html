Tips for vim

Vim-slime:

  • Turning vim into a interactive script runner
  • Vim-slime
  • Install:
    • add Plug 'jpalardy/vim-slime' in vim-plug
  • Usage:
    • To use with neovim terminal, add let g:slime_target = "neovim" to .vimrc.
    • Default shortcut: <C-c><C-c> to send this paragraph (vip) to terminal.
    • To use with tmux, add the following code to .vimrc:
    • let g:slime_target = "tmux"
    • let g:slime_paste_file = "$HOME/.slime_paste"
    • For more details, go to .vimrc and Shelbyvimrc.vim.

Tmux:

  • Solve for the cursor shape not changing when switch mode:
    • add set -g -a terminal-overrides ',*:Ss=\E[%p1%d q:Se=\E[2 q' to $HOME/.tmux.conf
  • Solve for lag when switch mode:
    • add set ttimeoutlen=100 in $HOME/.vimrc.
    • add set-option -sg escape-time 0 to $HOME/.tmux.conf

Auto wrap text:

  • autocmd BufRead,BufNewFile *.md,*.tex setlocal textwidth=80 fo-=l fo+=t
  • if vim read an old / create a new markdown / tex file, then
    1. set text width as 80
    2. ensure t option is added => wrap text by text width
    3. delete l option => long lines are also wrapped.
  • press gqq to manually wrap a line, gqip for a paragraph.
